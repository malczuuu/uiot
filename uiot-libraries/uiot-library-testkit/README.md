# uIoT Library Testkit

A shared library containing test utilities and base classes for integration testing uIoT microservices. This library
leverages Testcontainers to provide reusable components for setting up test environments.

## Usage

This library is included as a dependency in all uIoT microservices.

```kotlin
dependencies {
    implementation(project(":uiot-libraries:uiot-library-testkit"))
}
```
